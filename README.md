# 🌦 Weather Application

A responsive weather application that fetches real-time weather information using the OpenWeather API.

## Features

- Search weather by city
- Live temperature
- Humidity
- Weather description
- Weather icons
- Error handling for invalid cities

## Technologies Used

- HTML
- CSS
- JavaScript
- Fetch API
- OpenWeather API

## How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/weather-app.git
```

2. Open

```
a.html
```

in your browser.

## API Used

OpenWeather API

## Concepts Used

- JavaScript Fetch API
- Async/Await
- DOM Manipulation
- REST APIs
- Error Handling
